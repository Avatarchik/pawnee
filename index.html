Pawnee is a menubar utility designed to help you manage your local Apache installation.
With Pawnee, always keep at hand your Apache settings. Easily enable modules, edit your virtual hosts or switch between multiple PHP versions by using Brew, everything without quitting your current activity.
Download now 1.0.0 - Requires OSX 10.10

OSX integration

The utility manages the default Apache software shipped with your Mac, and integrates seamlessy with OSX Yosemite. Just launch the app, fill your sudo password and start monitoring your installation.

Auto restart

Do you need to edit your httpd.conf file by yourself to match your specific needs ? Work and save the file by using your favorite text editor, and the server will automatically restart to retrieve your updated configuration.

Virtual hosts and modules

Easily view and edit your configured virtual hosts, and manage enabled modules. Your httpd.conf file will be updated automatically.

Brew support

Did you installed multiple PHP versions by using the Homebrew-php repository ?
The available versions are displayed and you can switch to the one you want with a single click.

Activity

Easily monitor what is going on with your installation by reading the activity panel. All operations and commands are listed there.

Security

Did you configure things by yourself in your httpd.conf file ? Pawnee will handle it and only modify the needed code, without messing around with your custom data.
The tool also makes a backup before doing destructive operations.
